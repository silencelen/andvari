andvari browser extension — install (household build, loaded from disk)
========================================================================

Before you start: the extension talks to the household server over Tailscale,
so this computer must be on the tailnet (same as the desktop app).

Keep the unzipped folder somewhere permanent (e.g. Documents\andvari-extension).
The browser loads the extension FROM that folder — if you delete or move it,
the extension breaks.

Chrome / Edge / Brave
---------------------
1. Unzip this file to a permanent folder.
2. Open the extensions page:
     Chrome: chrome://extensions    Brave: brave://extensions    Edge: edge://extensions
3. Turn ON "Developer mode" (toggle in the top-right corner; in Edge it's
   in the left sidebar).
4. Click "Load unpacked" and pick the unzipped folder.
5. Pin andvari (puzzle-piece icon > pin), click it, and sign in with your
   email + master password. Unlocking takes a few seconds - that is the
   key-stretching doing its job.

Firefox
-------
Firefox only keeps unsigned extensions loaded until you close the browser
(a store-signed version may come later). To load it for a session:
1. Unzip this file.
2. Go to about:debugging#/runtime/this-firefox
3. Click "Load Temporary Add-on..." and pick manifest.json inside the folder.
4. After restarting Firefox you'll need to repeat steps 2-3.

Updating (already installed?)
-----------------------------
1. Download the new zip and unzip it OVER the same folder you used before
   (replace everything).
2. Open the extensions page and click the reload (circular arrow) icon on
   the andvari card. Firefox: redo the temporary load.
3. Reload any open tabs you want andvari active in.

Using it
--------
- Just installed or updated? Reload any tabs that were already open — the
  page part of andvari loads with the page.
- Click into a username or password field on a login page: a small andvari
  dropdown lists your matching logins; click one to fill both fields.
- Submit a login the extension doesn't know: a "Save to andvari?" bar
  appears at the top of the page.
- The toolbar popup shows lock status; "Lock" wipes the keys from memory.
- Nothing is stored on disk unencrypted - closing the browser locks it.

Problems? Note the site + what you expected, and send it with the other
test feedback.
